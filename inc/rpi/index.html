A Raspberry Pi For A Good Idea

I'm giving away two Raspberry Pi's per month, for free, to anybody which has a good use for it. It includes a case, an SD card and a USB cable. Shipping included to anywhere in the world. However, there are a few conditions, which are listed below.

Submit Idea Vote on Ideas More info

Rules

  • You have to write down your idea and publish it to Gitlab (or Github). It doesn't have to be code, a README.md is good as well. It has to be in English.
  • You have to post your idea to the voting platform on uservoice, including a link to the Gitlab or Github project.
  • The two ideas with the most votes will get the Raspberry Pi sent the month after.
  • You can only win once every 3 months.
  • If you have a software project, the code must be released under the GPL v2 or later.
  • I'll track your progress and create a page/article about your project on Raymii.org

The bundle includes:

  • Raspberry Pi, Model B:
    • 700 MHz ARM11 (ARMv6) CPU)
    • 512 MB RAM
    • 100 Mbit Ethernet (RJ-45)
    • 2x USB 2.0
    • 1x HDMI/1x Composite RCA
    • SDHC Card slot
    • 8x GPIO, UART, I2C Bus, SPI
  • USB cable (for power)
  • Case (transparant)
  • SD Card (4 or 8 GB)
